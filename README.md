# Maison Verre Analytics Portfolio

**A living analytics case study built on a fictional luxury retail brand.**

Maison Verre is a boutique luxury retailer — the fictional scenario powering every 
analysis in this repository. Rather than isolated exercises, every query, dashboard, 
and workbook here answers a real business question: How is the brand performing? 
Where are clients being lost? Which months are margin risks? What does leadership 
need to know?

This repository documents a growing body of work across SQL, spreadsheet analytics, 
and data visualization — built to the standard expected in luxury retail, fashion, 
and consumer analytics roles.

---

## What's Inside

### SQL Track 
**15 tasks + capstone | PostgreSQL | DB Fiddle**

Structured query progression from foundational SELECT statements through advanced 
multi-CTE window function queries. All work uses the Maison Verre schema — a 
`clients` and `sales` table architecture reflecting real luxury retail data relationships.

Skills demonstrated:
- Filtering, aggregation, and GROUP BY logic
- JOINs across relational tables
- Subqueries and Common Table Expressions (CTEs)
- Window functions: RANK(), ROW_NUMBER(), PARTITION BY
- Multi-CTE capstone: Client Intelligence Report (four result sets)

---

### Google Sheets Track 
**15 tasks + capstone | Google Sheets**

End-to-end spreadsheet analytics progression covering dynamic formulas, dashboard 
design, and cohort analysis. All work uses the Maison Verre monthly performance 
dataset across nine KPIs.

Skills demonstrated:
- VLOOKUP, INDEX/MATCH, dynamic named ranges
- Conditional formatting and traffic-light logic
- Sparklines and KPI dashboard design
- IFERROR, AVERAGEIF, RANK, LARGE
- Cohort retention matrix with triangular boundary logic
- Capstone: four-sheet Executive Analytics Report including a VP-ready 
  Executive Summary with formula-driven highlights and analyst narrative

---

## The Maison Verre Scenario

Maison Verre is a fictional single-location luxury boutique operating in a 
competitive prestige retail environment. The brand serves a high-value client 
base where Average Transaction Value, Sell-Through Rate, and client retention 
are the metrics that separate a thriving season from a margin crisis.

Every analysis in this repository is grounded in that context — not generic 
datasets, but business questions a real luxury retail analyst would be asked 
to answer.
